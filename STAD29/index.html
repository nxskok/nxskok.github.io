  • 2024-03-13 16:30: before I forget, from today’s class, loading MASS like this:

library(MASS, exclude = "select")

loads the whole of MASS except for the one troublesome function select that gets tangled up with the tidyverse select, and if you load MASS this way, you don’t have to worry about the conflicted stuff. (You can also use include to say “load only these functions and no others”, which is in the same spirit as the Python from x import y).
